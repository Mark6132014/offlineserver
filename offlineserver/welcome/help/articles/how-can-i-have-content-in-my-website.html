How Can I Have Content In My Website

In OfflineServer

Making Content In Your Website Is Quite Simple, If You Know How To Do These Skills:


You need to learn these three programming languages:


The Three Programming Languages You Should Know About First:


HTML - HyperText Markup Language


CSS - Cascading StyleSheets


JS - JavaScript


If you know these three programming languages, you are in luck, if you do not know, go to this website, and learn HTML, CSS, and JS:


W3Schools
W3Schools HTML
W3Schools CSS
W3Schools JS

If you are offline like the purpose of OfflineServer, you can download or view this PDF file! It may be long, but worth it!


SamsTeach Yourself HTML, CSS, and JavaScript by Julie C. Meloni (608 pages)

If you learned about these three programming languages (or already), you can start making your free website in OfflineServer!


Put the files you want to have in a folder that you can remember the path of that folder. And start writing


Once you are finished with your code, make sure to save it. Next go to this link: http://admin.offlineserver.net/editor and log in with


your credentials, then go to settings > import file


There will be a input saying that it needs the file path to import it, type it in, and make sure instead of the back slashes, put in forward slashes.


If not, your file ain't going to make it to your website. Once you imported all of your files, are are all set now! Now you can make a hostname for your website or test it!
